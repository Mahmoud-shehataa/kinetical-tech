# KINETICAL TECH Website

Static responsive bilingual website package.

## Files
- `index.html` — page markup and bilingual content.
- `styles.css` — responsive styling, brand colors, accessibility states.
- `script.js` — language toggle, mobile menu, reveal animation.
- `assets/` — extracted Kinetical brand assets used in the site.
- `PROMPT.md` — implementation prompt used for the build.

## Run locally
Open `index.html` in a browser, or serve the folder with any static server.

Example:
```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Notes
The site uses the approved English and Arabic Kinetical content, the provided brand colors, official Kinetical visual assets, and responsive/mobile UX rules from the attached UI/UX skill.
